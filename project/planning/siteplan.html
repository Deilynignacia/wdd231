A Christmas Carol: The Musical - Digital Stage Manager

Site Planning

SITE NAME

Site Name: Digital Stage Manager

Description: The website will be a resource management tool for the cast and production team, one that will only possible to access using a password.

Optional domain availability:ccmcast.org

SITE PURPOSE

The purpose of the "Digital Stage Manager" is to serve as source of information for the entire cast and production team of "A Christmas Carol - The Musical.". This site will provide essential services such as facilitate immediate and organized access to the script, musical scores and tutorials and offer a Dashboard with relevant announcements, rehearsal schedules, and production-team contact information.

SCENARIOS

Questions that site visitors might ask:

  • Where is the fastest place to find only the resources related to my character, and how can I ensure I am downloading the most recently updated score?
  • Is there a section on the main page that tells me if any rehearsals were cancelled or delayed

COLOR SCHEMA

A color that evokes a festive and traditional Christmas atmosphere.

  • Primary Color (Christmas Red): #821616 - Will be used for key section backgrounds, important accents, and decorative borders.
  • Secondary Color (Dark Green): #2e422c - Will be used for content section backgrounds, headings, and some elements.
  • Accent Color (Gold/Ivory): #A79573 - Will be used for borders, decorative lines, links, and text accents.
  • Background Color (Cream/Light): #F8F4ED - Will be ued for the general site background.
  • Text Color (Dark): #1B1B1B - Will be used for the main text in paragraphs and subheadings (Almost black).

TYPOGRAPHY

The font selection aims to balance readability with a Christmas aesthetic.

  • Headings (H1, H2, H3): "Playfair Display", serif. Chosen for its elegant and classic style that aligns with the Victorian aestethic of the musical. It will be used for the main headings and titles.
  • Body Text (Paragraphs, Lists): "Open Sans", sans-serif. Selected for its readability. It will be use for long blocks of text.
